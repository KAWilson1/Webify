What

Webify.py is a that tool reads from your notes and transforms them into an HTML file (like the one you are looking at right now).

Webify.py:

  • is easy on the eyes
  • increases readability
  • increases access

    • Why

    Notes can often be sloppy and poorly formatted.

    Manually converting notes to HTML is time consuming.

    How

    Webify.py reads from a text file called notes.txt

    It then separates your notes into blocks and styles those blocks randomly

    Lastly it compiles your webified document using HTML and CSS

    Using Webify.py

    The title block can have any number of lines displayed within

  • The end block must be followed with an empty line
  • There cannot be blank lines anywhere else in the document

    • The following blocks must take the following structure

  • Header1
  • Detail1
  • Subdetail1
  • Subdetail2
  • Detail2
  • Header2

    • Ensure that notes.txt is in the same folder as webify.py

    Run webify.py and your output will be called webified.html
